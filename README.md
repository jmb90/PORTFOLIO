# Data Analysis Portfolio

This repository contains practical exercises and projects developed as part of my learning journey in data analysis and business intelligence.

The goal of this portfolio is to demonstrate my progress, skills, and understanding of different data tools through hands-on practice.

## Repository Structure

Each folder represents a tool or technology I am currently learning or have worked with:

- **Excel**  
  Data cleaning, analysis, formulas, and pivot tables.

- **SQL**  
  Basic to intermediate queries, joins, and data analysis.

- **Power BI**  
  Dashboards, data modeling, and visualizations.

Each folder includes exercises, datasets, and brief explanations where applicable.
